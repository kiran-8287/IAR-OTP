## ✅ Tasks Completed

- 🎨 Built a Sign-In Page with OTP Verification using only HTML, CSS, and JavaScript
- 🖌 Clean & Responsive UI  
  - Yellow-white-black theme with subtle orange highlights  
  - Inspired by IIT Palakkad’s design aesthetics
- 🏗 Functional Layout Includes:
  - 📧 Email input field
  - 📨 “Send OTP” button — generates a 6-digit OTP (dummy flow, logs to browser console)
  - 🔢 OTP input field
  - 🔓 “Login” button — validates OTP and shows success/failure message
- ✨ UI Features:
  - Smooth transitions & hover effects
  - Card layout with rounded inputs
  - Fully mobile responsive using only vanilla code
👉 Redirect user to profile page after successful login (for future features)




## ⚠ Issues Faced

- 📱 Detailed adjustments needed for UI spacing across mobile & desktop breakpoints
- 🦉 Balancing accessibility and readability with a yellow-heavy theme was challenging
- 🎨 Fine-tuned button styling for contrast without overusing orange



## 🚀 Suggestions for Improvement

- 📧 Integrate EmailJS or NodeMailer to send real OTPs via email
- 🛡 Add input validation for proper email formatting & OTP length
- 🔄 Display loading spinner/real-time feedback when “Send OTP” is clicked
